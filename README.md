# OutPathOptionsMod
The modification provides some (mostly cheat) settings that allow you to customize some aspects of the game (OutPath) and improve your gameplay.

# Install
1. Install BepInEx 5.4.23.5 [BepInEx Releases](https://github.com/BepInEx/BepInEx/releases) [How to install](https://docs.bepinex.dev/articles/user_guide/installation/index.html)
2. Install this plugin [Releases](https://github.com/VitaminkaR/OutPathOptionsMod/releases)

# Features
- Infinite stats (stamina, food, hp)
- Instant interact (no delays between clicks)
- Instant break resources
- Instant crafting
- Giving credits
- Dupe items in hand
- Pick up all items on map
- Automatic pickup items
- Infinite builds battery
- Building work radius multiplier
- Building work speed multiplier
- Mining buildings instantly destroy props
- Destroy all resource nodes
- Kill all entities on the map
